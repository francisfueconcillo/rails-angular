    Ruby on Rails with AngularJS Scaffolding¶ ↑

    Features¶ ↑

    • Manage front-end dependencies separately using Bower. bower-rails

    • Includes Bootstrap-sass, Angular-Bootstrap, LoDash

    • Partial HTMLs are automatically converted to $templateCache and added to assets pipeline. angular-rails-templates

    System dependencies¶ ↑

    • Ruby version: 2.2.0

    • Rails version: 4.2.0

    • Install NodeJS

    • Install Bower

    npm install bower -g

    Specifications¶ ↑

    • Uses sqlite3 by default for Active Record. Change this to preferred database engine

    How to run¶ ↑

    1. Clone this repo. Install all dependencies.

    2. Edit Gemfile to add more gems.

    3. run bundle install, to install rails dependencies

    4. Edit Bowerfile to add more front-end dependencies. Components will be located at /vendor/assets/bower_components

    5. run rake bower:install, to intall front-end dependencies

    6. run rails s

    7. Open your browser at localhost:3000. You should see a smaple AngularJS app.
